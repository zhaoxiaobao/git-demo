

# 兼职系统

## 项目介绍

兼职系统是一个为学生和社会人士提供灵活就业机会的平台。我们致力于连接雇主和兼职人员，为他们提供一个简单、高效的交流和合作平台。

## 主要功能

- 雇主可以发布兼职工作岗位，包括工作时间、地点、薪资等信息。
- 兼职人员可以浏览并申请感兴趣的兼职工作。
- 雇主可以查看兼职人员的简历和申请情况，并进行筛选和面试。
- 兼职人员可以接受或拒绝雇主的工作邀请。
- 系统提供消息通知功能，方便雇主和兼职人员之间的沟通。
